# 01 — Iconographie : le seul « langage » du jeu

> Tout le jeu découverte se joue avec les symboles ci-dessous. **On ignore tout le texte.**
> Ces icônes sont **déjà imprimées** sur les vraies cartes de la boîte.

## Les 3 symboles que tout le monde doit connaître

| Symbole | Nom officiel | Ce que ça veut dire au stand | Où le lire |
|---|---|---|---|
| **★ (étoile)** | Recruit | **« argent »** : sert à **acheter** un héros | en bas à gauche d'une carte héros |
| **griffure** (trois traits / « scratch ») | Attack | **« coup »** : sert à **frapper** (et à scanner) | en bas à gauche d'une carte héros |
| **chiffre seul, coin bas-droit** | Cost | **« prix »** : combien de ★ pour acheter cette carte | coin inférieur droit d'une carte héros |

C'est **tout** ce qu'il faut pour jouer. Les trois phrases du stand :
> ★ = j'achète · griffure = je tape · le chiffre en bas à droite = le prix.

## Les autres icônes utiles

| Symbole | Sens | Détail |
|---|---|---|
| **chiffre à côté d'une griffure sur un ENNEMI** | PV de l'ennemi | il faut dépenser **autant** de griffure pour le tuer |
| **chiffre sur une carte STRIKE** (liseré rouge) | dégâts | autant de **PV perdus** quand un ennemi te touche |
| **Health** (cœur / chiffre sur l'Avatar) | points de vie du joueur | tu es K.O. si tes dégâts cumulés ≥ ce chiffre |
| **Time Track** (piste 10 → 0 sur le plateau) | minuteur global | arrive à 0 = tout le monde perd |
| **☎ (téléphone)** | sortie de la Matrix | utilisé seulement par la variante « Real World ↔ Matrix » |
| **liseré coloré + petite icône en haut à gauche** | Classe du héros | **on l'ignore** (sert aux combos textuels) |

## Anatomie d'une carte (vraie carte Legendary: The Matrix)

```
  ┌───────────────────────────────┐
  │ [icône classe]   TITRE (texte) │  ← ignoré
  │                  NOM PERSO     │  ← ignoré
  │                                │
  │        [ ILLUSTRATION ]        │
  │                                │
  │  ★ 2   [texte capacité]        │  ← on lit SEULEMENT "★ 2"
  │  griffure ?           ┌─────┐  │
  │                       │  3  │  │  ← COÛT = 3 étoiles pour l'acheter
  │                       └─────┘  │
  └───────────────────────────────┘
```

- **En haut à gauche** : icône de classe + couleur de liseré → **ignorés**.
- **Au milieu** : du texte de capacité → **ignoré** (« le texte ne compte pas »).
- **En bas à gauche** : la (les) icône(s) de ressource → **★** et/ou **griffure** = ce que la carte rapporte.
- **En bas à droite** : le **coût** (un chiffre) = prix d'achat en ★.

### Carte ENNEMI (liseré rouge, « Enemy »)
- Une seule valeur qui compte : **la griffure + son chiffre** = PV à lui retirer pour le tuer.
- ⚠️ Quelques ennemis affichent un **◎** à la place du chiffre = « ne peut pas être combattu » (on les
  écarte du set découverte, voir `04-selection-cartes.md`).

### Carte STRIKE (liseré rouge sombre, « Strike »)
- Un **chiffre = dégâts**. On le pose près de l'Avatar du joueur touché. Cumul ≥ Health → K.O.

### Carte AVATAR (le perso qu'on incarne)
- **Health** = PV (seul chiffre utile au stand). *Rank* (ordre du tour) et *Speed* = ignorés.

## Les cartes « monnaie » de départ (essentielles)

La boîte contient les briques de base, parfaites car **quasi sans texte** :

- **« Unplug »** (35 ex.) → carte **★** de base (Recruit). = l'argent de départ.
- **« There Is No Spoon »** (25 ex.) → carte **griffure** de base (Attack). = le coup de départ.
- **« Free Your Mind »** (5 ex.) → carte à effet (capacité d'Avatar, **texte**) → **écartée** du jeu cœur.

## Anti-sèche à plastifier pour le stand (brouillon)

```
   ★  = ACHETER        (étoile)
  ╱╱╱ = FRAPPER        (griffure)   ← scanne aussi
   3  = PRIX (en bas à droite)
  ENNEMI : son chiffre = coups pour le tuer
  STRIKE : son chiffre = PV que tu perds
  ⛔ Le texte ne sert à rien. Regarde les symboles.
```
